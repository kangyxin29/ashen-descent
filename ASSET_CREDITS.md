# Asset Credits

This document tracks all external assets used in the Ashen Descent project, including their sources and license terms.

---

## Principles

1. **Only use assets with permissive licenses** (CC0, permissive CC, or explicitly reusable)
2. **Limit sources to 2-3 consistent packs** for visual cohesion
3. **Document everything** with original license terms
4. **Consistency over variety** - unified style matters more than diverse sources

## How to Add Assets

When adding new assets:

1. Verify the license allows commercial/reusable use
2. Document the source URL
3. Note the license name and any conditions
4. Ensure the asset matches our unified style guide

### Style Guidelines

- **Tile size**: 16x16 or 32x32 pixels
- **Color palette**: Dark fantasy with ember accents (#ff6b35)
- **Perspective**: Top-down, consistent light source
- **UI framing**: Matching border radius, glow effects
